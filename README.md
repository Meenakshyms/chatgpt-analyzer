# ChatGPT Analyzer

A web application that analyzes ChatGPT responses and provides structured feedback.

## Features
- Input ChatGPT responses
- Analyze clarity, tone, and relevance
- Simple web interface

## Tech Stack
- Python (Flask)
- HTML, CSS

## How to Run
1. Install dependencies
2. Run `python app.py`
3. Open `http://localhost:5000`
